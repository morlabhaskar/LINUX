Shared Memory is the fastest Inter-Process Communication (IPC) mechanism in Linux.
It allows multiple processes to access the same memory region to exchange data without kernel.
Types of Shared Memory in Linux
System V Shared Memory (shmget, shmat)
POSIX Shared Memory (shm_open, mmap)
