In Linux, a thread is the smallest unit of execution within a process.
A process can have multiple threads, and all threads of a process share the same resources.
Memory Sharing in Threads
All threads share:
    Code segment
    Global variables
    Heap
    File descriptors
Each thread has:
    Own stack
    Own registers
    Own thread ID (TID)
getpid();   // returns process ID
gettid();   // returns thread ID
